## 웹 (client)

### 채팅 안읽음 표시 버그픽스 (`TeamChatTab.tsx`)

- **버그**: 내가 보낸 메시지도 안읽음 구분선(N개 안 읽은 메시지) 위에 표시되는 문제
- **원인**: `firstUnreadIndex`에서 `lastSeenAt` 이후 메시지 전체를 탐색 → 내 메시지도 포함
- **수정**: `findIndex` 조건에 `m.userId !== myUserId` 추가

  ```ts
  const firstUnreadIndex = useMemo(() => {
    if (!lastSeenAt) return -1;
    return messages.findIndex(
      (m) => new Date(m.createdAt) > lastSeenAt && m.userId !== myUserId
    );
  }, [messages, lastSeenAt, myUserId]);

  const unreadCount = useMemo(() => {
    if (firstUnreadIndex < 0) return 0;
    return messages.slice(firstUnreadIndex).filter((m) => m.userId !== myUserId).length;
  }, [messages, firstUnreadIndex, myUserId]);
  ```

## Flutter (apps/)

### 채팅 안읽음 뱃지 기능 추가

#### 1. `UnreadService` 신규 (`apps/lib/core/unread_service.dart`)

- 방별 미읽음 카운트를 앱 실행 중 메모리에서 관리하는 싱글톤
- `ValueNotifier<Map<String, int>>` 기반 → UI 자동 재빌드 지원

  ```dart
  UnreadService.increment(roomId)  // FCM 수신 시 증가
  UnreadService.clear(roomId)      // 채팅방 입장 시 초기화
  UnreadService.countFor(roomId)   // 현재 카운트 조회
  ```

#### 2. FCM 포그라운드 메시지에서 카운트 증가 (`notification_service.dart`)

- `FirebaseMessaging.onMessage` 리스너에서 `data['roomId']` 추출 후 `UnreadService.increment` 호출

  ```dart
  FirebaseMessaging.onMessage.listen((message) {
    final roomId = message.data['roomId'];
    if (roomId != null) UnreadService.increment(roomId);
    // 로컬 알림 표시 (기존과 동일)
  });
  ```

#### 3. 채팅방 목록에 안읽음 뱃지 표시 (`team_screen.dart`)

- 방 목록 `ListTile` → `ValueListenableBuilder` 로 감싸 미읽음 카운트 반응
- 카운트 > 0이면 파란색 원형 뱃지, 0이면 기존 화살표/삭제 아이콘 유지

#### 4. 채팅방 진입 시 카운트 초기화 (`room_chat_screen.dart`)

- `_ChatTabState.initState()`에서 `UnreadService.clear(widget.roomId)` 호출
