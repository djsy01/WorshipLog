# 2026-05-06 작업 내역

## 오늘 반영된 주요 기능 요약

이전 작업에서 미커밋 및 추가 예정이었던 주요 기능(악보 구조 개편, 사용 가이드, 찬양 권한/정렬 등)과 UI/UX 개선 사항들을 모두 코드에 반영 및 정리했습니다.

### 1. 콘티 악보 다중 업로드 및 구조 개편

- **DB 스키마 마이그레이션**
  - 기존 `conti_songs.sheet_music_url` 단일 악보 체제에서 `conti_song_sheets` 테이블을 통한 다중 악보 체제로 변경.
- **기본 악보 / 임시 악보 분리 적용**
  - **기본 악보**: 찬양 자체에 귀속 (`songs.sheetMusicUrl`)
  - **임시 악보**: 특정 콘티에만 종속 (`conti_song_sheets`)
- **악보 뷰어 및 PDF 출력 개선**
  - 다중 임시 악보가 등록된 경우 순서대로 렌더링 및 개별 삭제 기능 제공.
  - 임시 악보가 없으면 기본 악보가 자동으로 표시(Fallback)되도록 원본 보호.
  - PDF 저장 시 다중 악보가 모두 순서대로 출력되도록 레이아웃 반영.

### 2. 찬양 관리 고도화

- **관리자 역할(Role) 시스템 추가**
  - 유저 스키마에 `role` 추가 (일반 유저는 `user`, 관리자는 `admin`).
  - 찬양 수정/삭제 시 일반 유저는 본인이 등록한 곡만 제어 가능하고, 관리자는 전체 곡 제어 가능.
- **정렬 및 페이지네이션**
  - 찬양 목록을 가나다 및 알파벳 순(`orderBy: { title: 'asc' }`)으로 정렬.
  - 한 페이지당 20곡(`PAGE_SIZE = 20`)씩 표시되도록 페이지네이션 구현 (검색, 카테고리 변경 시 리셋).
- **찬양 상세 모달 도입**
  - 기존 카드의 인라인 펼침 UI를 제거하고 화면 중앙에 뜨는 전용 상세 모달 UI로 교체.
- **찬양 수정 모달 — 악보 업로드 기능**
  - 모달 내에서 찬양 기본 악보(PDF/이미지)를 바로 업로드, 확인, 삭제할 수 있는 파일 관리 UI 추가.

### 3. 대시보드 사용 가이드 페이지 추가

- `/dashboard/guide` 라우트 신규 생성.
- **찬양 관리, 콘티 작성, 팀스페이스, 말씀 묵상** 4가지 핵심 기능에 대한 단계별 안내 제공.
- 아코디언 접기/펼치기 방식 적용 및 섹션별 테마 컬러(Orange, Violet, Blue, Green) 부여.

### 4. 기타 UI/UX 개선 및 버그 수정

- **AppHeader Hydration 오류 해결**: SSR 환경과 클라이언트의 로컬 스토리지 접근 시점 차이로 발생하는 불일치(Mismatch) 현상 수정.
- **오버스크롤 배경 최적화**: 브라우저 오버스크롤 시 노출되는 `html` 배경색을 라이트/다크 모드 환경에 맞춰 이질감 없이 적용.
- **PDF 인쇄 깜빡임 개선**: 인쇄 모드 종료 시(`afterprint`) 다크모드로 복귀할 때 발생하는 화면의 하얀 깜빡임 현상 최소화.
