# 상현 강도사님 피드백 정리

1. 찬양 추가를 콘티 구성 피드에서도 할 수 있으면 좋을 것 같다
   (콘티를 짜는데 거의 찬양이 없어서 계속 찬양 검색 피드와 콘티 피드를 왔다갔다 해야하는 불편함이 있음)

2. PDF 공유를 눌러보니 직접 악보 사이즈를 줄이지 않으면 한 페이지 안에 들어가지 않는 문제점
   (기존 악보가 A4 꽉차는 크기이면 송폼과 악보가 동시에 한 페이지에 들어가지 않음)

3. 송폼 폰트가 너무 작음
   (인도자나 싱어가 보기에 불편함이 있을 것 같음)

4. 벌스, 프리코러스, 코러스, 브릿지 등
   싱어들 혹은 세션이 확인할 수 있도록 악보 작업을 할 수 있는 페이지도 만들수 있으면 만드는 것도 좋을듯 해요!!
